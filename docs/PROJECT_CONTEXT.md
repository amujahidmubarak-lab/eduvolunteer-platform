# Project Context: Malang Mengajar

## Overview
Malang Mengajar is a human-centered educational volunteer management platform designed for university social service programs.

## Purpose
- Manage volunteers
- Organize teaching schedules
- Monitor educational social impact
- Simplify organizational operations
- Support sustainability of educational service programs

**Note:** This is NOT a complex LMS.

## Tech Stack
- Laravel 13
- Tailwind CSS 4
- Alpine.js
- PostgreSQL (via Supabase)
